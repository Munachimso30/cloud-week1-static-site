About this project

Static site on Amazon S3

This project is a simple static website built with plain HTML files. The files are stored in an Amazon S3 bucket that has static website hosting enabled, so S3 can serve index.html and other pages directly.

Public access with bucket policy

By default, S3 buckets are private. To make this site publicly readable, I disabled the “Block all public access” setting for this bucket and added a bucket policy that allows the s3:GetObject action for everyone on all objects in the bucket. This makes the HTML files accessible over the internet, but still read-only.

CloudFront in front of S3

A CloudFront distribution sits in front of the S3 website endpoint. CloudFront caches my pages at edge locations around the world and serves them over HTTPS. This improves performance and security compared to using the raw S3 website URL.
